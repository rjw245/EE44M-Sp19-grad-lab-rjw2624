118 // String draw function.
119 // 16 rows (0 to 15) and 21 characters (0 to 20)
120 // Requires (11 + size*size*6*8) bytes of transmission for each character
121 // Input: x columns from the left edge (0 to 20)
122 // y rows from the top edge (0 to 15)
123 // pt pointer to a null terminated string to be printed
124 // textColor 16-bit color of the characters
