54 // hardware connections
55 // **********ST7735 TFT and SDC*******************
56 // ST7735
57 // Backlight (pin 10) connected to +3.3 V
58 // MISO (pin 9) unconnected
59 // SCK (pin 8) connected to PA2 (SSI0Clk)
60 // MOSI (pin 7) connected to PA5 (SSI0Tx)
61 // TFT_CS (pin 6) connected to PA3 (SSI0Fss)
62 // CARD_CS (pin 5) unconnected
63 // Data/Command (pin 4) connected to PA6 (GPIO), high for data, low for command
64 // RESET (pin 3) connected to PA7 (GPIO)
65 // VCC (pin 2) connected to +3.3 V
66 // Gnd (pin 1) connected to ground
67 
68 // **********wide.hk ST7735R with ADXL345 accelerometer *******************
69 // Silkscreen Label (SDC side up; LCD side down) - Connection
70 // VCC - +3.3 V
71 // GND - Ground
72 // !SCL - PA2 Sclk SPI clock from microcontroller to TFT or SDC
73 // !SDA - PA5 MOSI SPI data from microcontroller to TFT or SDC
74 // DC - PA6 TFT data/command
75 // RES - PA7 TFT reset
76 // CS - PA3 TFT_CS, active low to enable TFT
77 // *CS - (NC) SDC_CS, active low to enable SDC
78 // MISO - (NC) MISO SPI data from SDC to microcontroller
79 // SDA � (NC) I2C data for ADXL345 accelerometer
80 // SCL � (NC) I2C clock for ADXL345 accelerometer
81 // SDO � (NC) I2C alternate address for ADXL345 accelerometer
82 // Backlight + - Light, backlight connected to +3.3 V

87 // GND - Ground

92 // CS - PA3 TFT_CS, active low to enable TFT
